# Troubleshooting

## No audio after starting

- Reload the media tab.
- Stop and start enhancement again.
- Try System Default output.

## Crackling or pitch drift

- Use System Default output first.
- Close heavy tabs.
- Avoid repeated rapid preset switching during long playback.
- Reopen Studio only when detailed control is needed.

## Output device list incomplete

Use the explicit unlock flow in the popup or Studio output selector.
