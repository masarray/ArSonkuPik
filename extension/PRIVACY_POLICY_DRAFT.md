# ArSonKuPik Privacy Policy Draft

ArSonKuPik is a Chrome extension that enhances audio from a browser tab selected by the user.

## Data handled

ArSonKuPik handles the selected tab's audio stream locally in the browser using the Web Audio API. The extension may also store local extension settings, presets, and optional output-device preferences using Chrome extension storage.

## How data is used

Captured tab audio is used only to provide the extension's audio enhancement features, including EQ, compression, analog color, stereo width, limiting, meters, and optional output routing.

## Local processing

ArSonKuPik does not record audio to files, upload audio, transmit audio to servers, sell user data, share user data, or use data for advertising.

## Output-device routing

When the user explicitly selects `Unlock device list…`, the extension may request local audio-device access for the extension origin so Chrome can list available output devices. This is used only to route processed audio to a selected speaker/headphone device. ArSonKuPik does not record microphone audio.

## Data sharing

ArSonKuPik does not share user data with third parties.

## Limited Use statement

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## Contact

For privacy questions, contact the developer through the contact information listed in the Chrome Web Store developer profile.
