Local mastering chain for Chrome tab audio

Make browser audio feel richer, wider, and more alive.

ArSonKuPik is a Chrome extension that enhances active-tab audio locally with precision EQ, compressor, analog-style color, multiband stereo width, limiter, smart presets, and a premium Studio view.

Built like a compact mastering console.

Every module supports the single purpose: better local tab audio, with a clean UI and Web Store-friendly privacy posture.

MasAri smart presets

Balanced presets for music, audiophile listening, open-air character, podcast, movies, and night listening.

Analog 4-band color

Adds low punch, warm body, pleasant presence, and silky high-frequency sparkle without harsh distortion.

Phase-aware width

Low and vocal ranges stay controlled while high-side particles get a tasteful stereo lift.

Fluid visual engine

Spectrum and meters use smoothing to feel responsive while keeping CPU and memory usage lightweight.

Output routing

System Default uses a direct route; optional device routing is unlocked only by explicit user action.

Chrome Store ready

Manifest V3, no host permissions, no content scripts, no remote executable code, and clear privacy documentation.

Professional public release posture

The repository includes source code, documentation, Web Store packaging scripts, GitHub Release automation, GitHub Pages deployment, privacy policy, support page, and wiki pages. Store submission remains manual by design.
