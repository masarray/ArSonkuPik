Support

Need help with ArSonKuPik? Use this page as the public support URL for Chrome Web Store listing.

Before reporting an issue

  • Reload the target media tab.
  • Stop and start enhancement from the popup.
  • Confirm the browser is Chrome 116 or newer.
  • Try System Default output first before testing a specific output device.

Useful details for reports

  • Chrome version
  • Operating system
  • Website where the issue happens
  • Selected preset
  • Whether custom output routing is active
  • Steps to reproduce

Known limitations

Browser audio processing depends on Chrome tab capture and Web Audio runtime behavior. Some protected, muted, or unusual audio pages may behave differently.
