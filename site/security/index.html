Security

ArSonKuPik is designed as a local-processing Chrome extension with no remote DSP, no host permissions, no content scripts, and no analytics pipeline.

Release safety

  • GitHub Releases include a Web Store ZIP, source archive, and SHA-256 checksums.
  • Chrome Web Store submission is manual.
  • CRX packaging is intentionally disabled.
  • Private keys and store API credentials must never be committed.

Report a vulnerability

Please use the contact channel listed on the support page and include reproduction steps, affected version, and expected impact.
