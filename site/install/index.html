Install guide

The recommended installation channel is Chrome Web Store. Release ZIP files are for maintainers and testers before Web Store submission.

For regular users

Install from Chrome Web Store after the listing is approved. Users do not need source code or release ZIP files.

For testers

  1. Download the Web Store upload ZIP from a GitHub Release.
  2. Extract it locally.
  3. Open chrome://extensions.
  4. Enable Developer mode.
  5. Choose Load unpacked and select the extracted folder.

Distribution policy

This project does not publish CRX files. The public release automation builds a clean Web Store ZIP and a source archive only.
