Usage workflow

  1. Open a media tab.
  2. Click ArSonKuPik.
  3. Click Start Enhance.
  4. Open Studio for deeper module control.

Preset philosophy

MasAri is the balanced default. Audiophile emphasizes refined detail. Punchy Music adds body and punch. Open Air is designed for bigger sound while keeping phase safer than full-band widening.

Troubleshooting

If audio behaves unexpectedly, stop enhancement, reload the media tab, and start again from the popup. Prefer System Default output for the most stable route.
