Privacy Policy

Effective date: 2026-06-23

ArSonKuPik enhances active browser-tab audio locally on the user's device.

Data collection

ArSonKuPik does not collect, sell, share, or transmit personal data.

Audio handling

Audio is processed locally in the browser using the Web Audio API. The extension does not record audio, upload audio, store audio files, send audio to a server, sell audio data, or share audio data with third parties.

Local settings

The extension may store local preferences such as presets, audio settings, A/B snapshots, and UI preferences using Chrome extension storage.

Output device access

Output device access is used only for local output routing after the user explicitly unlocks or selects an output device.

Remote code and analytics

ArSonKuPik does not load remote executable code and does not use analytics tracking.

Contact

For privacy questions, use the support contact provided in the Chrome Web Store listing or the project repository.
