# Chrome Web Store Upload Package

The Chrome Web Store upload package is generated by:

```bash
npm run package:webstore
```

The generated ZIP is placed in `release/` and is the only file intended for manual upload to the Chrome Web Store Developer Dashboard.

## Included files

```text
manifest.json
popup.html
studio.html
offscreen.html
src/
icons/
```

## Excluded files

```text
docs/
site/
wiki/
scripts/
.github/
release/
*.md
tools/
node_modules/
*.pem
*.key
*.crx
font binaries
```

## Review posture

The extension is designed with a narrow single purpose: local enhancement of browser tab audio.

The runtime package does not include remote executable code, host permissions, content scripts, analytics SDKs, or cloud upload logic.
