# Privacy Policy

Effective date: 2026-06-23

ArSonKuPik is a Chrome extension that enhances audio from the active browser tab locally on the user's device.

## Data collection

ArSonKuPik does not collect, sell, share, or transmit personal data.

## Audio handling

ArSonKuPik processes tab audio locally in the browser using the Web Audio API.

The extension does not:

- record audio,
- upload audio,
- store audio files,
- send audio to a server,
- sell audio data,
- share audio data with third parties.

## Local settings

The extension may store user preferences locally using Chrome extension storage, including presets, audio settings, A/B snapshots, and UI preferences. These settings stay on the user's browser unless the browser itself synchronizes extension data according to the user's Chrome settings.

## Output device access

ArSonKuPik may request local audio output device access only when the user explicitly chooses to unlock or select an output device. This is used for local output routing only.

## Remote code and analytics

ArSonKuPik does not load remote executable code and does not use analytics tracking.

## Contact

For privacy questions, use the support contact listed on the project page or Chrome Web Store listing.
